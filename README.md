# [雁不过衡阳的二次开发CobaltStrike](https://github.com/Yang0615777/SecondaryDevCobaltStrike)
## 关于二次开发后的CobaltStrike</br>
默认的CobaltStrike内存在多处流量特征,在马儿与服务端建立连接时进行流量交互.此间存在多处可疑特征已被各大杀软记录在册</br>
如:卡巴斯基,诺顿,迈克菲等，但国内杀软并无此功能.</br>
